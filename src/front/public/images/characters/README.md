# Персонажі / фракції для фону

Тут можна додати власні зображення персонажів гри — вони з’являться на фоні головної та інших сторінок як декоративні силуети.

**Файли (за бажанням замініть SVG на PNG):**
- `undead.png` — Нежить
- `waterfolk.png` — Водний народ  
- `imperials.png` — Імперійці
- `highlanders.png` — Гірці

Якщо додасте файли з такими іменами (наприклад `undead.png`), оновіть шляхи в `app/components/GameBackground.tsx` (замініть `.svg` на `.png` для потрібних персонажів), щоб використовувались ваші картинки замість placeholder SVG.

Рекомендовані пропорції: висота більша за ширину (наприклад 120×200 px або більше), прозорий або темний фон.
